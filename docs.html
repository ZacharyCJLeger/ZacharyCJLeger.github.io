How to Install

To install, simply select your operating system and click the download button. The executable should download and you shoud be able to run it directly on your computer.

Layout

This is the program you will see upon load-up (without the green numbers, of course). Here's is a basic run-down of what you are seeing:

  1. This is the options menu button. There, you will see a list of options for delete elements in your space. This button is for convenience, you can also do this via commands.
  2. This is the command line history. Here, you will see the history of commands you executed.
  3. This is the 3D visualization space. Here, you will see the drawings you create. By clicking on it, you can use your mouse, the WASD keys, the space button and the control button to move around the 3D environment.
  4. This is the command bar. Here, you can type in your commands. Pressing enter will cause them to execute.
  5. Here is the Send button. Instead of pressing enter to send your commands, you can press this instead.
  6. This section is your environment function section. Here, the functions (and variables) you create are stored and displayed. More details on this section below.

Variables in GeoQuack are simply functions without parameters (i.e. constant functions). To create your first function, try typing [a] := e‚ÇÅ+e‚ÇÄ into the command bar (the values e1 and e0 will automatically convert to e‚ÇÅ and e‚ÇÄ upon typing). Manipulating Individual Functions

For each function in the environment function section, there may be up to three options:
  1. The drop-down button. Clicking this button will display more details on the value of the function
  2. The draw button. This will attempt to draw the function if it of a drawable type. Only constant functions (i.e. variables) are drawable.
  3. The delete button. This will delete the function from the environment.

Creating Drawings

As described in the previous section, after creating a function, you can draw the function (if it is a constant function) by clicking the draw button in the function environment section. However, this option always stores the resulting drawing in a constant function with an auto-generated name. You can create drawings in the command line by calling 
[drawing_name] := draw e‚ÇÅ+e‚ÇÄ

Once you create your drawings, you can move around the 3D space by first clicking on the 3D environment on the right-hand side to enter the environment. Then, by holding and dragging the right mouse button, you can move the camera to look around. To move around, you can use the WASD keys to move forward, left, back and right respectively. You can also use the space key to move up and the left shift key to move down.
